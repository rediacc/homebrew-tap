# Homebrew Tap for Rediacc

This is the official Homebrew tap for [Rediacc](https://www.rediacc.com) tools.

## Installation

```bash
brew tap rediacc/tap
brew install rediacc-cli
```

Or install directly:

```bash
brew install rediacc/tap/rediacc-cli
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `rediacc-cli` | Rediacc CLI - automation and scripting tool |

## Updating

```bash
brew update
brew upgrade rediacc-cli
```

## Uninstalling

```bash
brew uninstall rediacc-cli
brew untap rediacc/tap
```
